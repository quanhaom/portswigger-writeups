# PortSwigger Web Security Academy Writeups

This repository contains my learning notes and security writeups based on PortSwigger Web Security Academy labs.

## Objectives

* Improve web application security knowledge
* Understand OWASP Top 10 vulnerabilities
* Practice security testing methodologies
* Learn secure coding and remediation techniques

## Categories

* SQL Injection
* Cross-Site Scripting (XSS)
* Cross-Site Request Forgery (CSRF)
* Access Control
* Authentication
* Server-Side Request Forgery (SSRF)
* File Upload Vulnerabilities
* XXE
* Deserialization
* Business Logic Vulnerabilities

## Disclaimer

These notes are intended for educational purposes and focus on vulnerability analysis, impact assessment, and remediation concepts.
